# Firmware- July 8th
Firmware includes: 
- Wi-Fi connection on boot
- NTP sync
- 12/24-hour display toggle
- Alarm on/off toggle
- Alarm triggering: checks each second whether the current time matches the alarm time (only when alarm is enabled and not snoozed).
- Pulsing Buzzer sound: instead of a continous tone the buzzer beeps in short bursts. 
- Stop alarm
- Snooze alarm
- Menu: 5 item menu (set set alarm hour, set alarm minute, toggle 12/24h, toggle alarm on/off, exit) navigated with up/down and confirmed with OK

Estimated Time: 1-1.5 hours
